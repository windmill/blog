INSTALLATION
============
1) Download Vesper
2) Unpack the zip file
3) Upload the 'vesper' folder and all of its contents to the '/wp-content/themes/' directory
4) Activate the theme through the 'Design' menu in WordPress
5) Download the wp-pagenavi (http://wordpress.org/extend/plugins/wp-pagenavi/) plugin by Lester Chan
6) Activate the plugin through the 'Plugins' menu in WordPress
7) Done, enjoy!
